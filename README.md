# Karagümrük Hentbol Website Starter

Bu paket GitHub Pages'e doğrudan yüklenebilir.

## İçerik
- Türkçe / İngilizce dil geçişi
- Ana sayfa
- Haberler
- Maç bölümü
- Takım bölümü
- Akademi ve başvuru formu
- Üyelik şartları ve başvuru formu
- Bağış bölümü
- Vitorra resmi spor giyim partneri bölümü
- İletişim
- Admin paneli tasarım önizlemesi

## GitHub'a yükleme
1. Repository içindeki eski dosyaları yedekleyin.
2. Bu klasördeki tüm dosyaları repository köküne yükleyin.
3. Commit changes seçin.
4. GitHub Pages 1-3 dakika içinde siteyi günceller.

## Önemli
`admin.html` şu anda yalnızca arayüz önizlemesidir. Gerçek giriş, veri kaydetme, fotoğraf yükleme ve içerik yayınlama için sonraki aşamada Supabase bağlantısı yapılmalıdır.
