KARAGÜMRÜK HENTBOL STATIC WEBSITE

1) Cloudflare Pages'te "Upload your static files" seçeneğine girin.
2) Bu ZIP dosyasını bilgisayarınızda çıkarın.
3) Çıkan klasörün içindeki tüm dosyaları seçip Cloudflare'a yükleyin.
4) Deploy / Publish düğmesine basın.
5) Size verilen .pages.dev adresini açın.

ÖNEMLİ:
- Ana dosyanın adı index.html olarak kalmalıdır.
- Takım bilgilerini index.html içinden değiştirebilirsiniz.
- Renkler ve görünüm styles.css içindedir.
- Menü ve haber geçişleri script.js içindedir.
- İletişim formu şu anda demo şeklindedir. Gerçek mesaj göndermek için Formspree gibi ücretsiz bir servis bağlanmalıdır.
